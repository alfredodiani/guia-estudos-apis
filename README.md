## Como contribuir

Contribuições são bem-vindas. Fluxo simples sugerido:

1. Fork o repositório.
2. Crie uma branch com um nome descritivo: `feature/nome-da-feature` ou `fix/descricao`.
3. Faça alterações e abra um Pull Request com descrição clara do que foi alterado.

Abra uma issue para discutir mudanças maiores antes de implementar.
